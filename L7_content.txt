Features of ReactJs
1. JSX
HTML-like syntax inside JavaScript to build UI easily.
2. Components
Reusable pieces of UI (like functions for UI).
3. Unidirectional Data Flow (One Way Direction)
Data flows in one direction (parent → child), making apps predictable.
4. Virtual DOM
Fast UI updates by changing only what is needed.
5. Simplicity
Easy to learn, clean structure, less code.

6. Performance
Efficient rendering makes apps fast.

Integrate React with Spring Boot application ||
• Cross-Origin Resource Sharing (CORS) is a security mechanism implemented by web browsers to restrict
cross-origin HTTP requests. It prevents web pages from making requests to a different domain than the one
from which the page was served.

• To resolve CORS issues, you can follow these steps:
1. Configure CORS in your Spring Boot backend:
• Add the @CrossOrigin annotation to your controller class or specific controller methods to allow crossorigin requests from the React frontend. @CrossOrigin(origins = "*")
2. Handle CORS in your React frontend:
• Install the cors package (npm install cors) to add a middleware that allows cross-origin requests in your
development server.
